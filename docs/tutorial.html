Input Map

Key Binding Action Name Description
W S A D Movement Move Forward, Backward, Left, and Right.
SPACE Player Jump Perform an Ollie to clear obstacles.
SHIFT Player Run Sprint or Boost speed while skating.
LMB or H Player Hook Fire grappling hook to swing or grab objects.
I Inventory Open your backpack to view collected items.
E Use Active Item Interact with the world or use a power-up.
